# 说明
使用大语言模型翻译STM32 HAL库 标准库，并人工校对

- 文本模型：glm-5
- 切片大小：10KB-20KB
- 切片数量：
- - HAL-F4 G4 H7 1979

希望这能帮助你快速学习和开发STM32
