# 说明

使用大语言模型翻译STM32 HAL库 标准库中注释和补充必要注释，并人工校对



翻译模型：GLM5



切片大小：10KB-20KB



更新：

- 2026.5.5 HAL F4 G4 H7 1979

- 2026.5.10 HAL F0 F3 F7 1315



好用给个Star，没用骂个Issue



希望这能帮助你快速学习和开发STM32



不定期更新

# STM32 官方库来源

本项目翻译所使用的官方库均来自 STMicroelectronics 官方渠道：

## HAL / LL

- STM32CubeF1: https://github.com/STMicroelectronics/STM32CubeF1
- STM32CubeF4: https://github.com/STMicroelectronics/STM32CubeF4
- STM32CubeG0: https://github.com/STMicroelectronics/STM32CubeG0
- STM32CubeL4: https://github.com/STMicroelectronics/STM32CubeL4
- STM32CubeH7: https://github.com/STMicroelectronics/STM32CubeH7
